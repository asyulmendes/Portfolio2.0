## 🛠️ Tecnologias Utilizadas

* **HTML5** (Estrutura semântica)
* **CSS3** (Estilização avançada e Media Queries)
* **JavaScript** (Lógica do menu mobile e interatividade)
* **Google Fonts** (Tipografia personalizada)

## 🎨 Diferenciais do Projeto

* **Design Responsivo:** Adaptado para smartphones, tablets e desktops.
* **Estética Cyber/Gamer:** Uso de cores neon e efeitos de vidro (Glassmorphism).
* **Foco em UX:** Navegação intuitiva e botões com feedback visual (hover).
* **Branding Pessoal:** Integração da minha identidade como desenvolvedora e criadora de mundos.
